### Мультиклассовая классификация отзывов
Необходимо исследовать различные методы работы с текстом для реализации классификации отзывов магазина на 3 класса - магазин, доставка, товары. Метрика для оценки - f1.

### Multiclass review classification
It is necessary to investigate various methods of working with text to implement the classification of store reviews into 3 classes - store, delivery, goods. The metric for evaluation is f1.
